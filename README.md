## Usage ğŸ”§

```bash
drdir <URL> <Wordlist> [Options]
```

- `<URL>`: The URL of the site you wish to scan.
- `<Wordlist>`: A file containing a list of paths to scan.
- `[Options]`: Additional options (such as the number of threads).
